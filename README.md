# Space-Dodger
CMSC 137 1st1617 CD-4L Project

* To play the game:

	Server
		1. cd GameCore
		2. java GameServer

	Player
		1. cd GameCore
		2. Change string host in Main.java to server's current IP address
		3. Compile then run Main
		4. Enter username

* There must be 3 players to start the game.
